This folder contains driver of this IP.

make.sh : make the IP driver
rt_mergetree: the driver folder, which follows the java.util.Arrays namespace
rt_mergetree/java/util/make.sh: make the IP driver for c and java wrapper
rt_mergetree/java/util/Arrays.java: the java.util.Arrays wrapper for IP
rt_mergetree/java/util/sortlib.c: the IP driver library
